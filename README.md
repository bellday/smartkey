## 프로젝트 목적

* 음주운전 사고를 우리가 제작한 음주측정 스마트키로 사전에 방지한다.

## 조원

이원균, 조은영, 최종일, 하성혁
* 이원균 : 차량 SW 개발
* 조은영 : 기술 조사, 부품 선정
* 최종일 : 알고리즘 순서도 작성, 차량 하드웨어제작
* 하성혁 : 차량 SW개발
  
## 개발 환경

* Arduino IDE
* Fritzing
